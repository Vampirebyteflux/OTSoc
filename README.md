## Overview
OTSoc is a modern Security Operations Center (SOC) dashboard designed to visualize, monitor, and manage security events in a centralized interface. The project focuses on providing a clean, interactive, and realistic SOC experience suitable for demonstrations, hackathons, and learning purposes.

---

## Features
- Real-time security dashboard visualization  
- Alert and threat monitoring interface  
- Incident management view  
- Log monitoring panel  
- Threat/attack visualization UI  
- Structured security reports section  
- Cinematic intro and smooth UI animations
